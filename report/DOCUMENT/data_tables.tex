Στους παρακάτω πίνακες αναγράφεται το πλήθος των πράξεων που \tl{AABB-AABB} 
και τρίγωνο-τρίγωνο που εκτελέστηκαν για κάθε σενάριο.
Ο συμβολισμός \textbf{\tl{O.T.}} αναφέρεται στον 
αλγόριθμο που χρησιμοποιεί ένα \tl{sKD-Tree (One Tree algorithm)}.
O συμβολισμός \textbf{\tl{T.T.}} αναφέρεται στον 
αλγόριθμο που χρησιμοποιεί δύο \tl{sKD-Tree (Two trees algorithm)}.
